📚 StudyMate – Student Task Management System

StudyMate is a web-based task management application designed to help students organize their academic workload efficiently. Built with Flask, the platform allows users to manage subjects, track tasks, and monitor study progress through an intuitive dashboard.

🚀 Features
🔐 User Authentication

Secure user signup and login system

Personalized dashboard for authenticated users

Guest landing page with clear call-to-action

📘 Subject Management

Users can manage their study subjects with custom attributes:

Add subjects with:

Subject name

Target study hours

Custom color coding

View subjects in a grid layout

Track progress with:

Progress bars

Task completion percentage

Completed vs total tasks

Delete subjects easily

✅ Task Management

Create and organize tasks linked to subjects.

Each task includes:

Task title

Associated subject

Due date

Estimated study time

Completion status

Users can:

Mark tasks as complete/incomplete

Delete tasks

View tasks categorized by subject color

📊 Progress Tracking

StudyMate provides visual insights into academic progress.

Subject-level progress bars

Task completion statistics

Real-time updates of:

Active subjects

Total tasks

Completed tasks

🎨 User Interface

Clean and modern design

Responsive layout

Custom CSS styling

Visual indicators such as:

Color-coded subjects

Progress bars

Status badges

Intuitive forms with validation

Helpful empty-state messages

🧠 Dashboard Layout

The dashboard is organized into two panels:

Left Panel

Subject management

Subject overview

Right Panel

Task management

Task tracking

This layout helps students easily connect tasks with their corresponding subjects.

⚙️ Installation
1️⃣ Clone the repository
git clone https://github.com/aarthi-kumari/studyMate.git
cd studyMate
2️⃣ Create virtual environment
python -m venv venv

Activate it:

Windows

venv\Scripts\activate

Mac/Linux

source venv/bin/activate
3️⃣ Install dependencies
pip install -r requirements.txt
4️⃣ Run the application
python app.py

Open in browser:

http://127.0.0.1:5000
